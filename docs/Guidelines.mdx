# Design System Guidelines

This documentation provides guidelines for using tokens, components, and theming.

## Tokens

- Tokens are defined in `src/tokens` and built via `pnpm run tokens:build`.

## Theming

- Use the `ThemeProvider` from `src/theme/ThemeProvider` to enable runtime switching.

## i18n

- Translations live in `src/locales/en` and `src/locales/bn` and are loaded via `src/i18n`.
